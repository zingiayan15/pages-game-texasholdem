# pages-game-texasholdem

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以德州扑克相关功能或演示为主，可作为独立文件直接使用或引用。

## 项目简介

这是一个轻量级的页面归档仓库，集中存放了一批针对德州扑克游戏场景的 HTML 文件。页面设计为自包含结构，不依赖外部框架或后端服务，适合快速预览、本地运行或嵌入其他项目。

## 目录说明

- `pages/`：存放所有 HTML 页面文件，每个文件为一个独立功能页面。
- `assets/`（如有）：存放页面引用的静态资源，如样式表、脚本、图片等。
- `README.md`：本文件，仓库说明文档。

页面文件命名遵循功能描述原则，便于识别和检索。

## 页面归档说明

所有页面均为纯前端实现，数据逻辑在浏览器端完成。页面之间无强依赖关系，可单独打开使用。

归档内容包括但不限于：
- 游戏规则展示与说明
- 牌型概率计算或模拟
- 简易记分或状态记录工具
- 教学或演示用途的交互页面

页面可能使用现代浏览器特性（如 ES6、CSS Grid 等），建议使用最新版主流浏览器访问。

## 维护说明

- 本仓库持续归档新的页面文件，但不对旧页面进行频繁更新。
- 页面内容仅供参考学习或非商业用途，使用者可根据需要自行修改。
- 不保证所有页面在所有浏览器或设备上完全兼容。
- 提交 issue 或 pull request 时请说明具体页面及问题描述。

## 使用方式

1. 克隆或下载本仓库。
2. 在浏览器中直接打开 `pages/` 目录下任意 HTML 文件。
3. 也可将页面部署到任意静态文件服务器上使用。

无需安装依赖或配置环境。
